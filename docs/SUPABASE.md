## Tables (Main Project)

### `buy_price_history`
Stores per-unit buy prices extracted from supplier invoices.

| Column | Type | Description |
|---|---|---|
| id | uuid | Primary key |
| item_name_raw | text | Original invoice description |
| item_name_hub | text | Mapped Hub item name |
| supplier | text | Supplier name |
| invoice_ref | text | Invoice/order reference |
| invoice_date | date | Invoice date |
| unit_price_ex_gst | numeric | Price per unit (bottle/can) ex-GST |
| qty_units | numeric | Total units in this line |
| created_at | timestamptz | Insert timestamp |

### `soh_reports`
Monthly stock-on-hand snapshots (auto-generated by cron).

| Column | Type | Description |
|---|---|---|
| id | uuid | Primary key |
| snapshot_date | date | Snapshot date (last day of month) |
| items | jsonb | Array of `{ name, category, onHand, buyPrice, value }` |
| total_value | numeric | Total stock value |
| created_at | timestamptz | |

### `specials`
Bar specials for the specials display board.

| Column | Type | Description |
|---|---|---|
| id | uuid | Primary key |
| name | text | Special name |
| description | text | Description |
| price | numeric | Price |
| image_url | text | Supabase storage URL |
| active | boolean | Show on display board |
| created_at | timestamptz | |

### `documents`
Metadata for files saved to OneDrive (invoices, POs, reports).

| Column | Type | Description |
|---|---|---|
| id | uuid | Primary key |
| type | text | `invoice` / `purchase_order` / `report` |
| supplier | text | Supplier name |
| po_ref | text | PO reference |
| filename | text | OneDrive filename |
| onedrive_path | text | Full OneDrive path |
| invoice_date | date | |
| created_at | timestamptz | |

### `sessions` (Roster)

| Column | Type | Description |

| end_time | time | |

### `session_volunteers` (Roster)

| Column | Type | Description |
|---|---|---|
